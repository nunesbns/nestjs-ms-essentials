## Description

This is a [Nest](https://github.com/nestjs/nest) framework start project for microservices purposes.

## What includes

- Schema Validation using `Joi`, you can validate env variable out of the box.

## Roadmap

I am planning to add following features:

- ~~Enviroment validation~~
- Logging Interceptor to log Request/Responses
- Caching
- Migrations
- Unit and e2e tests
- Authorization
- Kafka Consumer/Producer

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
